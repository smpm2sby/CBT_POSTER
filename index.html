🚀 3 Langkah Mudah, Kamu Siap Ujian!

1

Siapkan Perangkatmu!

Aplikasi Ujian Android

CBT EXAM

Instal aplikasi resmi melalui Google Play Store.

Aplikasi Ujian iOS (iPhone)

Exambrowser

Instal aplikasi resmi melalui App Store.

2

Ambil Berkas Ujian!

Link Download Soal:

bit.ly/CBTEXAM26

Alur Android

01.

Buka Folder bernama "CBT SAT SPEMDA"

02.

Download semua file soal mapel ujian

03.

Pastikan file sudah terdownload di HP

04.

Cek file di File Manager HP.

Alur iOS

01.

Buka Folder bernama "CBT SAT SPEMDA"

02.

Simpan file dari drive ke aplikasi "Files"

03.

Pastikan file sudah terdownload di HP

04.

Cek file di File Manager HP.

3

Memulai CBT!

Final Step

Masuk ke Wi-Fi "UJIAN"

Sebelum ujian dimulai, peserta wajib masuk ke Wi-Fi bernama UJIAN dan data seluler wajib dimatikan.

Cara Membuka Soal:

ANDROID:

Cari file .Cbt di File Manager, ketuk & buka dengan aplikasi CBT EXAM.

IOS (iPhone):

Buka aplikasi Files, ketuk file .Cbt, pilih Share & buka dengan Exambrowser.
